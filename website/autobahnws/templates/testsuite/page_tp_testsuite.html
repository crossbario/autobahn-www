Coverage

The {{ autobahntestsuite() }} employs a software testing technique known as fuzzing, runs fully automated and includes an extensive set of about 300 test cases covering

  • Framing
  • Pings/Pongs
  • Reserved Bits
  • Opcodes
  • Fragmentation
  • UTF-8 Handling
  • Limits/Performance
  • Closing Handshake
  • Opening Handshake (under development)
