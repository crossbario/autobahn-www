Installation Guide

This is a short guide how to install {{autobahntestsuite()}} on your system. For usage, please see the User Manual.

Prerequisites

{{ autobahntestsuite() }} is implemented in Python and Twisted. You need the following software installed:

  • Python
  • Python Setuptools
  • Twisted

The projects above provide easy to use installers or packages.

Recommended is using the latest Python 2.7 release. Python 2.6 will also work, but not Python 3 (currently).

Unless you know what you need / are doing, a 32-Bit Python is recommended.

Installation

{{ autobahntestsuite() }} is available on the Python package index.

To install, open a command shell and do:

{{ beginshell() }} easy_install autobahntestsuite {{ endshell() }}

Optionally, if you want to use secure WebSocket (WSS), you will need to have pyOpenSSL installed:

{{ beginshell() }} easy_install pyopenssl {{ endshell() }}

Testing

To test your installation, open a command shell and do:

{{ beginshell() }} wstest --help {{ endshell() }}

{{inlineshell('wstest')}} is the {{autobahntestsuite()}} executable and installed in a system-wide location, i.e. on Windows under

{{ beginshell() }} C:\Python27\Scripts\wstest.exe {{ endshell() }}

If {{inlineshell('wstest')}} cannot be found on your command line, make sure you have the Python Scripts path on your PATH.
