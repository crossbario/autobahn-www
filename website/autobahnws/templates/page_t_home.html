Open-source real-time framework for Web, Mobile & Internet of Things.
  1. With Autobahn you can build real-time enabled Web, mobile and embedded applications in no time.
  2. The Autobahn project provides open-source implementations of the WebSocket and WAMP protocols.
  3. WebSocket allows bidirectional real-time messaging on the Web and WAMP adds asynchronous Remote Procedure Calls and Publish & Subscribe on top of WebSocket.

  • add WebSocket connectivity to apps
  • add WAMP connectivity to apps
  • optimized for Android
