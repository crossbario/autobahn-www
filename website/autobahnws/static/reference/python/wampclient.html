Client¶

The classes autobahn.wamp.WampClientProtocol and autobahn.wamp.WampClientFactory are the base classes you derive from to implement WAMP clients.

Factory¶

class autobahn.wamp.WampClientFactory(url, debug=False, debugCodePaths=False, debugWamp=False, debugApp=False)¶

Twisted client factory for WAMP.

startFactory()¶

Called by Twisted when the factory starts up. When overriding, make sure to call the base method.

stopFactory()¶

Called by Twisted when the factory shuts down. When overriding, make sure to call the base method.

Protocol¶

class autobahn.wamp.WampClientProtocol¶

Twisted client protocol for WAMP.

call(*args)¶

Perform a remote-procedure call (RPC). The first argument is the procedure URI (mandatory). Subsequent positional arguments can be provided (must be JSON serializable). The return value is a Twisted Deferred.

onSessionOpen()¶

Callback fired when WAMP session was fully established. Override in derived class.

prefix(prefix, uri)¶

Establishes a prefix to be used in CURIEs instead of URIs having that prefix for both client-to-server and server-to-client messages.

Parameters:
  • prefix (str) – Prefix to be used in CURIEs.
  • uri (str) – URI that this prefix will resolve to.
publish(topicUri, event, excludeMe=None, exclude=None, eligible=None)¶

Publish an event under a topic URI. The latter may be abbreviated using a CURIE which has been previously defined using prefix(). The event must be JSON serializable.

Parameters:
  • topicUri (str) – The topic URI or CURIE.
  • event (value) – Event to be published (must be JSON serializable) or None.
  • excludeMe (bool) – When True, don’t deliver the published event to myself (when I’m subscribed).
  • exclude (list of str) – Optional list of session IDs to exclude from receivers.
  • eligible (list of str) – Optional list of session IDs to that are eligible as receivers.
subscribe(topicUri, handler)¶

Subscribe to topic. When already subscribed, will overwrite the handler.

Parameters:
  • topicUri (str) – URI or CURIE of topic to subscribe to.
  • handler (Python callable, will be called as in <callable>(eventUri, event).) – Event handler to be invoked upon receiving events for topic.
unsubscribe(topicUri)¶

Unsubscribe from topic. Will do nothing when currently not subscribed to the topic.

Parameters:topicUri (str) – URI or CURIE of topic to unsubscribe from.
class autobahn.wamp.WampCraClientProtocol¶

Simple, authenticated WAMP client protocol.

The client can perform WAMP-Challenge-Response-Authentication (“WAMP-CRA”) to authenticate itself to a WAMP server. The server needs to implement WAMP-CRA also of course.

authenticate(authKey=None, authExtra=None, authSecret=None)¶

Authenticate the WAMP session to server.

Parameters:
  • authKey (str) – The key of the authentication credentials, something like a user or application name.
  • authExtra (dict) – Any extra authentication information.
  • authSecret – The secret of the authentication credentials, something like the user password or application secret key.

:returns Deferred – Deferred that fires upon authentication success (with permissions) or failure.
