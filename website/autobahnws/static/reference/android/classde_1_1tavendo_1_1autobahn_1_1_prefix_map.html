Detailed Description

Mapping between CURIEs and URIs.

Provides a two-way mapping between CURIEs (Compact URI Expressions) and full URIs.

See also:
http://www.w3.org/TR/curie/
Todo:
Prefixes MUST be NCNames (http://www.w3.org/TR/1999/REC-xml-names-19990114/#NT-NCName)
Todo:
Work in the details of http://www.w3.org/TR/curie/ (default prefixes, ..)
