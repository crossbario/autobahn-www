Detailed Description

InputStream wrapping a ByteBuffer.

This class can be used i.e. to wrap ByteBuffers allocated direct in NIO for socket I/O. The class does not allocate ByteBuffers itself, but assumes the user has already one that just needs to be wrapped to use with InputStream based processing.
