Detailed Description

WebSocket writer, the sending leg of a WebSockets connection.

This is run on it's background thread with it's own message loop. The only method that needs to be called (from foreground thread) is forward(), which is used to forward a WebSockets message to this object (running on background thread) so that it can be formatted and sent out on the underlying TCP socket.
